# ☕ Java OOPS

This repository contains implementations of Object-Oriented Programming concepts in Java.

It covers core OOP principles with practical examples to build a strong understanding of object-oriented design and programming.

---

## 📌 Topics Covered

### Classes and Objects
- Class creation
- Object creation
- Accessing properties and methods

### Constructors
- Default Constructor
- Non-Parameterized Constructor
- Parameterized Constructor
- Constructor Overloading

### OOP Principles

#### Encapsulation
- Data hiding
- Getter and Setter methods

#### Inheritance
- Single Inheritance
- Multilevel Inheritance
- Hierarchical Inheritance

#### Polymorphism
- Method Overloading
- Method Overriding

#### Abstraction
- Abstract Classes
- Abstract Methods

#### Interfaces
- Interface implementation
- Multiple inheritance using interfaces

### Additional Concepts
- this keyword
- super keyword
- Static keyword

---

## 🎯 Purpose

This repository is created to:

- Understand Object-Oriented Programming deeply
- Strengthen Java development concepts
- Build strong coding fundamentals
- Practice real implementations of OOP concepts

---

## 💻 Structure

Each concept is implemented in separate Java files with clean and beginner-friendly examples.

---

## 🚀 How to Run

1. Clone the repository
2. Open in any Java IDE (VS Code / IntelliJ / Eclipse)
3. Compile and run the `.java` files

---

## 📚 Learning Approach

- Learn one concept at a time
- Understand theory
- Implement practical examples
- Push progress regularly to GitHub
